# NRI Cube Demo

This is a simple demo of rendering a colored spinning cube using [NRI](https://github.com/NVIDIA-RTX/NRI) in [Zig](https://ziglang.org/).

## Demo
![demo](assets/demo.png)

## Testing
Run the demo directly:
```sh
zig build run
```

## Building
Compile to `zig-out/bin`: 
```sh
zig build
```
